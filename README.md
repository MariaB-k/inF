# FASTA Reader & Sequence Analyzer

Эта работа представляет собой код на Python для чтения и анализа биологических последовательностей (нуклеотидных и белковых) в формате FASTA.

## Основные возможности:
- **Класс `Seq`**: хранит заголовок и последовательность, определяет длину и тип алфавита (нуклеотидный или белковый).
- **Класс `FastaReader`**: проверяет формат файла и читает его по строчкам.
- **Оптимизация**: использование генераторов (`yield`) позволяет быстро обработать файлы больших размера, не загружая много ин-фы в память устройства.

## Как запустить:
1. Укажите путь к вашему `.fasta` файлу в функции `demo()`.
2. Запустите скрипт: `python main.py`.
